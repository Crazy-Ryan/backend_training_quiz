停车信息表

| 字段名           | 字段类型 | 约束       | 存储内容             |
| ---------------- | -------- | ---------- | -------------------- |
| id               | int      | 主键       | 停车位id             |
| parking_lot_id   | char     | 非空       | 停车场编号           |
| space_no         | int      | 非空       | 停车场里对车位的编号 |
| license_plate_no | string   | 可为 NULL* | 车位上所停车辆车牌号 |

*当一个停车位上停车时，停车的车牌号为非空，所停车辆被取出后，车牌号变成空

初始化时需将每一个车位信息全部初始化，初始化时涉及**增**和**删**
停车和取车业务中仅涉及**改**和**查**两种业务，不涉及增和删
